manual :
the interrupt line should be configured and enabled in
order to generate an interrupt. This is done by programming 
the two trigger registers with the desired edge detection and 
by enabling the interrupt request by writing a ‘1’ to the
corresponding bit in the interrupt mask register. When the 
selected edge occurs on the interrupt line, an interrupt 
request is generated. The pending bit corresponding to the
interrupt line is also set. This request is cleared by writing 
a ‘1’ in the pending register.


manual :
For the configurable lines, an interrupt/event request can 
also be generated by software by writing a ‘1’ in the software 
interrupt/event register.

procedure:
1. Configure the corresponding mask bit in the EXTI_IMR 
register.
2. Configure the Trigger Selection bits of the Interrupt

3. Configure the enable and mask bits that control the NVIC 
IRQ channel mapped to the
EXTI so that an interrupt coming from one of the EXTI lines
